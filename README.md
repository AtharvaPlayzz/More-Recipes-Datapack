# More-Recipes-Datapack
More Recipes adds lots of new recipes for you to explore!
Including:
Tridents,
Enchanted Golden Apples,
Gilded Blackstone,
Netherite Blocks (from smithing together a diamond block with a netherite ingot),
Hearts of the Sea,
Saddles,
Shulker Shells,
Sponges,
Totems of Undying and
Chainmail Armor Set!
